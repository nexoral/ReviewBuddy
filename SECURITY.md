## Reporting a Vulnerability

We take the security of Review Buddy seriously. If you have discovered a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

### How to report

If you find a vulnerability, please do NOT open an issue. Instead, please report it privately:

1.  **GitHub Security Advisory**: Use the "Security" tab on the repository to report a vulnerability.
2.  **Email**: Send an email to [INSERT EMAIL ADDRESS] with details of the vulnerability.

### What to include

Please include as much information as possible to help us reproduce and fix the issue:
*   Steps to reproduce the vulnerability.
*   Example code or configuration.
*   The version of Review Buddy you are using.
*   Any relevant logs.

### Response

We will acknowledge your report within 48 hours and will aim to fix the vulnerability as soon as possible.
