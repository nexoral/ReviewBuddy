# Support

## How to get help

If you need help using Review Buddy, there are several ways to get support:

1.  **Documentation**: Check the [README.md](README.md) for usage instructions and configuration options.
2.  **GitHub Issues**: If you suspect a bug or have a feature request, check the [Issues](https://github.com/nexoral/ReviewBuddy/issues) page. If it hasn't been reported, open a new issue.
3.  **GitHub Discussions**: For general questions, ideas, or showing off your setup, join our [Discussions](https://github.com/nexoral/ReviewBuddy/discussions).

## Community

We want to build a friendly and helpful community. Please follow our [Code of Conduct](CODE_OF_CONDUCT.md) in all interactions.
